# Zobo Jobs X Post Explainer

## Style Block

Use the COLLINS identity from `DESIGN.md`: Canvas Parchment `#f8f8f7`, Absolute White `#ffffff`, Deep Onyx `#140700`, Pewter Accent `#cccccc`, Slate Echo `#5e5855`, and Shadowed Steel `#514c49`. Keep the system achromatic, editorial, and sharply typographic. Use Portrait Text / Times for large hooks and Graphik / Arial / system sans for operational detail.

## Format

X feed video, 1080x1080, 30 seconds, no voiceover required. Make the post understandable muted with large on-screen copy and fast visual rhythm.

## Rhythm Declaration

Pattern: `hook-fast-proof-build-PEAK-CTA`. Six scenes, each with a clear text beat and overlapping editorial transitions.

## Scenes

1. **Hook:** Recruitment agencies are losing margin on repeated first-round calls.
2. **Problem:** Calendars, phone screens, inconsistent notes, and slow client updates stack up.
3. **Shift:** Zobo Jobs runs structured AI interviews for every candidate.
4. **Workflow:** Invite candidates, AI interviews them, recruiter reviews scored evidence.
5. **Outcome:** Save cost and time while improving shortlist quality.
6. **CTA:** Move first-round interviews to AI. Book a Zobo Jobs demo.

## Recurring Motifs

Ledger lines, dossier cards, stamped labels, ranked candidate rows, proof ticks, and one dark onyx CTA close.

## Negative Prompt

Avoid colorful SaaS gradients, tiny text, generic AI robot tropes, and dense dashboard clutter. Keep every scene readable inside a scrolling X feed.
